  I/O口与外设：
  ========
  * 时钟：SYSCLK-180M  
	  APB1-45M，Timer-90M  
      APB2-90M，Timer-180M  

  *	CAN:  
    CAN1: PD0(RX), PD1(TX)      底盘电机1,2,3,4，抬升,7,8
    CAN2: PB12(RX), PB13(TX)    夹取1,2，抱柱3,4，云台5,6  
    
    * 底盘、抬升电机ID (CAN1)   

               抱柱机构（车头）  
          |\|= 1              2 =|/|

                == 7       8 ==   （抬升机构3508）

          |/|= 4              3 =|\|
               夹取机构（车尾）

    * 云台电机ID (CAN2)  

            -------------
                  | 
                 ( ) pitch 6
                -----
                 |_|  yaw 5

    * 夹取电机ID (CAN2)  

              |          |
              |          |
              |          |
           (1)------------(2)

    * 抱柱电机ID (CAN2)  

           ______     ______
           |    /     \    |
           |   (       )   |
           |   (3)   (4)   |
                       

  *	串口：  
    USART1:     PB7(RX)            DMA1-通道4数据流5  DR遥控器  
    USART3：	PD5(TX), PD6(RX)   DMA1-通道4数据流1  云台IMU  
    USART6：	PG9(RX), PG14(TX)  DMA2-通道5数据流1  裁判系统
  
  * GPIO：  
    PA0：    抬升霍尔 上限位  
    PA1：    抬升霍尔 下限位
	PA2：    抱柱气缸霍尔 抱紧
    * 霍尔

            -------------     最高点MAX
              |       |
              |       |
            -------------     气缸顶起位置MID
              |       |
            -------------     最低点MIN
             (_)     (_)
          
 
 *	气阀控制器：  
    0:    撑
    1：	  抱柱
    2：	  夹取
	3：	  弹弹药箱
	4：	  弹仓盖
	5：	  夹取伸缩
